# VR Bus Simulator
A WebXR experience that uses [A-Frame](https://aframe.io/) to simulate electric bus activity based on data from UTA.

## Setup
Host using:
```
$ npm install
$ npm start
```
Navigate to the locally hosted site on any VR headset connected to the same local network.
